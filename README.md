# My Neovim Config Files

---

## To use this configuration:

- Make sure you backup all your things:
    ```bash
    mv ~/.config/nvim ~/.config/nvim.back
    mv ~/.local/share/nvim ~/.local/share/nvim.back
    mv ~/.local/state/nvim ~/.local/state/nvim.back
    ```

- Copy the repo files:
    ```bash
    git clone https://github.com/peanutbutte-r/neovim-config.git ~/.config/nvim
    ```

- Run Neovim to install plugins:
    ```bash
    nvim
    ```
